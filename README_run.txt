=== Cara Menjalankan Aplikasi (Windows) ===

1) Taruh file berikut dalam satu folder:
   - app.py   (download dari Canvas ChatGPT)
   - requirements.txt  (file ini)
   - run_app.bat       (file ini)
   - (opsional) kopi_revisi.csv, jalan_revisi.csv

2) Klik dua kali run_app.bat
   Script akan membuat virtualenv, menginstal paket, dan membuka aplikasi di browser.

Jika terminal bilang 'streamlit' tidak dikenali, jalankan dengan python -m streamlit run app.py
